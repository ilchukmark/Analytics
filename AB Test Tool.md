# A/B Testing & User Segmentation Dashboard

**Project Overview:**
This interactive Tableau dashboard serves as a custom tool designed to analyze A/B test results and explore user segments. It provides a comprehensive view of experiment performance, making it easy to track variations and statistically evaluate outcomes.

**Key Features:**
* **Core Product Metrics:** High-level tracking of primary and secondary KPIs across control and test groups.
* **User Segmentation:** Deep dive into user demographics and behavioral traits to uncover segment-specific insights.
* **Distribution Analysis:** Visual representation of user allocation and metric splits between different test variants.
